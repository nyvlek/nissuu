Muestra tu catálogo de publicaciones alojadas en Issuu como una entrada en Wordpress usando un shortcode.

== Description ==
Issuu.com es una gran herramienta para alojar sus publicaciones en formato PDF, pero es mejor mantener a sus visitantes en tu sitio

Nissuu obtiene (a través de la API de Issuu) una lista de todos las publicaciones alojadas en tu cuenta issuu.com y permite mostrar esa lista en tu Wordpress a través de un sencillo shortcode, además puedes restringir opcionalmente la lista por etiqueta y controlar sus caracteristicas visuales.

== Installation ==
1. Descomprima el archivo zip
2. Copie los archivos en el directorio de su instalación de WordPress \"wp-content/plugins\"
3. Activar el plugin desde la pestaña \"Plugins\".
4. Vaya a Configuración Nyvlek Issuu, introduzca su \"API key\" y el \"API secret\". 
5. Utilice el código abreviado siempre que le parezca. Para personalizar su apariencia, utilice el archivo style.css CSS de su tema.

== Frequently Asked Questions ==
¿Cómo puedo utilizarla?
Agregue a una entrada o página la siguiente línea y listo:
[nissuu tag=\"\" viewer=\"mini\" titlebar=\"false\" vmode=\"\" ctitle=\"Haz click para ver publicación\" height=\"480\" bgcolor=\"FFFFFF\"]
